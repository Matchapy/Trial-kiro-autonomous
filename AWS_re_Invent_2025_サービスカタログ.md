## AI/機械学習

### Amazon Nova 2 Lite
高速かつコスト効率の良い軽量AIモデル。基本的な推論タスクに最適化されており、低レイテンシーを要求するアプリケーションに適しています。

### Amazon Nova 2 Pro
高度な推論能力を持つプロフェッショナル向けAIモデル。複雑なビジネスロジックや高度な意思決定プロセスに対応します。

### Amazon Nova 2 Sonic
音声対話AIに特化したスピーチ・ツー・スピーチモデル。自然でリアルタイムの会話型AIを実現し、業界最高水準の品質と価格性能比を提供します。

### Amazon Nova 2 Omni
マルチモーダルタスクに対応した汎用AIモデル。テキスト、画像、音声など複数のデータ形式を統合的に処理できます。

### Trainium3チップ
第4世代の自社開発AIチップ。前世代と比較して4倍の処理速度と4倍のメモリを実現し、トレーニングコストを50%削減します。

### Amazon EC2 Trn3 UltraServers
Trainium3チップを搭載した画期的なサーバーシステム。最大100万個のTrainium3チップを相互接続可能で、前世代の10倍のスケールを実現します。

### Frontier Agents (フロンティアエージェント)
数時間から数日間、人間の介入なしに自律的に動作できる新世代のAIエージェント。開発、セキュリティ、DevOpsの各領域で活用できます。

#### Kiro Autonomous Agent（Kiro自律エージェント）
チームの仮想開発者として機能するAIエージェント。コード作成、レビュー、テストを自律的に実行します。

#### AWS Security Agent（AWSセキュリティエージェント）
専任のセキュリティコンサルタントとして機能するAIエージェント。セキュリティリスクの検出と対応を自動化します。

#### AWS DevOps Agent（AWS DevOpsエージェント）
オンコール運用チームとして機能するAIエージェント。インシデント対応と運用業務を自動化します。

### Amazon Bedrock Open Training
顧客が独自のカスタムモデルをNovaモデルをベースに構築できる「オープントレーニング」サービス。組織固有のニーズに合わせたAIモデルの開発を可能にします。

### Amazon Bedrock AgentCore
高い信頼性を持つAIエージェントを構築するためのサービス。エージェントの境界管理、メモリ機能、評価機能を提供します。

## モダナイゼーション

### AWS Transform
メインフレームからWindows、VMwareシステムまで、レガシーシステムのモダナイゼーションを加速するエージェント型AIサービス。技術的負債を削減し、AIレディなテックスタックへの移行を支援します。

### AWS Transform Custom
組織固有のプログラミング言語やフレームワークを含む、あらゆるコード、API、フレームワーク、ランタイム、アーキテクチャの組織全体のモダナイゼーションを加速する新しいエージェント機能。

### AWS Transform Windows
.NET FrameworkアプリケーションとレガシーUIフレームワーク、SQL Serverデータベース、デプロイメントプロセスを含む、フルスタックのWindowsモダナイゼーションをAIエージェントで最大5倍高速化。クロスプラットフォームのオープンソース、クラウドネイティブソリューションへの変換を実現します。

**主な実績:**
- 推定11億行のコード分析
- 81万時間以上の手作業削減
- 既存の変換機能で4倍高速なモダナイゼーション

## セキュリティ

### IAM Access Analyzer 内部アクセス分析機能
組織内からのアクセスに関する包括的な可視性を提供する新機能。外部アクセス分析を補完します。

**主な特徴:**
- S3バケット、DynamoDBテーブル、RDSスナップショットへの内部アクセスの特定
- SCP、RCP、アイデンティティベースポリシーの自動推論による総合評価
- 統一ダッシュボードでの検出結果の集約
- Amazon EventBridgeとの統合による自動通知

**ユースケース:**
- 金融サービスや医療業界などの規制産業でのコンプライアンス対応
- クレジットカード情報や医療記録を含む機密データストアへのアクセス検証

### IAM MFA 強制適用
すべてのアカウントタイプのルートユーザーに対してMFA（多要素認証）を強制適用。パスワード関連攻撃の99%以上を防止します。

**主な特徴:**
- FIDO認証セキュリティキーのサポート
- FIDO2パスキーによるユーザーフレンドリーなMFA実装
- ルートユーザーおよびIAMユーザーごとに最大8つのMFAデバイスの登録が可能
- ハイパースケールクラウドプロバイダーとして初めて、すべてのルートアカウントで100% MFAカバレッジを達成

- 顧客とのやり取り中にアクションを推奨

#### 高度な機能
- 30以上の言語でのニューラル音声合成
- 自動音声認識
- AI駆動型カスタマーサービスサポートとアナリティクス
- リアルタイム処理

**ビジネスインパクト:**
- エージェントに「超人的な」能力を提供
- 顧客体験の大規模な変革
- カスタマーエンゲージメントの最適化

- 開発における柔軟性の提供

**ユースケース:**
- AIアプリケーションの信頼性確保

## まとめ

AWS re:Invent 2025では、AI/機械学習を中心とした革新的なサービスと機能強化が発表されました。特に、Frontier Agentsをはじめとするエージェント型AIの導入により、開発、セキュリティ、運用の各領域での自動化と効率化が大きく前進しています。

また、AWS Transformによるモダナイゼーションの加速、セキュリティ機能の強化、マルチクラウド対応の拡充など、企業のデジタルトランスフォーメーションを包括的にサポートする製品ポートフォリオが拡充されました。

これらの新サービスと機能強化により、AWSは引き続きクラウドコンピューティングとAIイノベーションのリーダーとしての地位を確立しています。
