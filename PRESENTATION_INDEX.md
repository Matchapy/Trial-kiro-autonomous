### Markdownから他の形式への変換

#### PowerPointへの変換
```bash
pandoc presentations/06_Frontier_Agents.md -o Frontier_Agents.pptx
```

#### PDFへの変換
```bash
pandoc presentations/06_Frontier_Agents.md -o Frontier_Agents.pdf \
  --pdf-engine=xelatex \
  -V CJKmainfont="Noto Sans CJK JP"
```

#### HTMLプレゼンテーション（reveal.js）
```bash
pandoc presentations/06_Frontier_Agents.md -t revealjs \
  -s -o Frontier_Agents.html \
  -V theme=black
```
